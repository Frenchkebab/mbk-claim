< 프로그램 사용방법 및 주의사항 >

1. 엑셀 파일의 CID넘버 오른쪽에 Memo 속성값이 자동으로 생깁니다.
  해당 값은 자동으로 마지막으로 완료한 작업을 기록하기 위해 사용되며,
  프로그램 재시작 시, 해당 값을 읽고 시작합니다.
  예) Memo: Vehicle-Logistics done 인 경우 -> query로 해당 CID값을 검색하여 Archive부터 시작

2. 페이지 로딩이 느린 경우, 최대 20분까지 기다립니다.
  20분이 넘어갈 경우 에러처리 후 다음 줄로 넘어갑니다.
  이 경우, 어디까지 완료되었는지 Memo에 기록됩니다.

3. 중간에 finished 표시가 되지 않은 파일들은, 다시 프로그램을 실행시킬 경우 알아서 나머지 작업들을 마무리합니다.


< chromedriver 파일 >

"Chrome:	97.0.4692.71" 과 같이 크롬의 version을 확인할 수 있습니다.

https://sites.google.com/chromium.org/driver/downloads 에 접속하여
해당 버전에 맞는 chromedriver.exe 파일을 폴더에 넣어주세요 (폴더의 가장 바깥에)


< 폴더 및 파일 설명>

[result]

프로그램 실행 결과를 출력한 파일들입니다.
프로그램시작시간.txt 로 저장되며, 얼마든지 지워도 되며 작업 기록을 확인하기 위해 생성됩니다.
문제 발생 시 해당 에러에 대한 기록을 남겨두었으니, 해당 파일을
tubowangin@gmail.com 으로 보내주세요.

[settings]

- login.txt
 : 로그인 정보(id, password)를 복붙하기 위한 파일이며 마음대로 수정해도 상관없습니다.

 - settings.txt
  : 프로그램 설정에 관한 파일이며, 콜론(:) 의 오른쪽 값을 수정하여 사용하면 됩니다.
    콜론(:)을 기준으로 오른쪽 값 (좌우공백 상관없음)을 인식합니다.

[upload]

: 작업에 사용될 파일들을 저장한 폴더입니다.
폴더 안에
- [BL]
- [LIST]
- [PICTURE]
- [(그 외 기타 폴더들)]
- xslx파일
처럼 넣고 사용하면 됩니다.
